## üì¶ Installation Guide

### 1. Download KSS2

Remove any **older versions** of KSS2 or **Galaxies Unbound** before installing.

Install the following into your `GameData` folder:

* `KSS2_Core.2.0.zip`
* Your selected systems (e.g. `SystemAlphaCentauri`, `SystemLuhman16`, etc.)
* *(Optional)* `ContractPack.2.0.zip` for career contracts

KSS2 supports **Volumetric EVE** for enhanced visuals. *(Legacy EVE is not compatible.)*

Your KSS2 folder should look like this:

## ‚öôÔ∏è Configuration Overview

Edit `KSS2Settings.cfg` in the KSS2 folder to customize your gameplay:

#### üîß Core Options

* `KerbolRevamp = False` ‚Äì *(upcoming)* revamp of stock system
* `HomeSwitch = True` ‚Äì set **Alva** as homeworld instead of Kerbin
* `DistanceFactor = 1` ‚Äì adjust system spacing (0.5, 1, 10, 100)
* `Rescale = 1` ‚Äì scale all systems *(requires Sigma Dimensions)*

#### üåç Expansions

* `PandorExpansion = True` ‚Äì adds the lush moon **Pandor** to Novin
* `HabitableProximaB = True` ‚Äì makes **Proxima b** a viable home

#### üõ∞Ô∏è Features

* `Wormholes = True` ‚Äì enable interstellar travel
* `KerbolWormhole = True` ‚Äì connects Kerbol ‚Üî Alpha Centauri
* `Anomalies = True` ‚Äì unlock hidden deep space features *(requires Blueshift)*
* `Exotics = True` ‚Äì enables rare, high-difficulty challenges
* `SunAsteroids / ExtraSolarAsteroids = True` ‚Äì asteroid belts

#### üì° CommNet

* `InterstellarCommNet = True` ‚Äì adds long-range relay coverage
